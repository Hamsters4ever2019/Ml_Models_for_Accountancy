# 📊 Accountancy ML Lab

An interactive Streamlit application demonstrating Machine Learning models for accounting and fraud detection.

## 🚀 Features
- **3 Supervised Models**: Random Forest, Logistic Regression, XGBoost
- **3 Unsupervised Models**: K-Means, DBSCAN, Isolation Forest
- **Synthetic Data Generator** for accounting transactions
- **Educational Tooltips** explaining each ML model
- **Interactive Visualizations** for model performance
- **File Upload** support for custom data

## 🛠️ Installation

```bash
git clone https://github.com/YOUR_USERNAME/accountancy-ml-lab.git
cd accountancy-ml-lab
pip install -r requirements.txt
streamlit run app.py
